# 🔥 Violence Detection using RWF-2000 Dataset

## 📌 Project Overview

This project focuses on detecting violent activities in videos using deep learning techniques.
The model is trained on the **RWF-2000 (Real-World Fighting Dataset)** to classify videos into:

* **Violence**
* **Non-Violence**

This system can be used in real-time surveillance, smart city monitoring, and security applications.

---

## 🎯 Objectives

* Detect violence in real-world video data
* Build a deep learning-based classification model
* Achieve high accuracy on unseen data
* Provide a scalable solution for surveillance systems

---

## 📂 Dataset

* Dataset: **RWF-2000**
* Source: Kaggle
* Contains real-world surveillance videos of violent and non-violent activities

🔗 Dataset Link:
https://www.kaggle.com/datasets/mohamedmustafa/real-world-violence-dataset

⚠️ Note: Dataset is not included in this repository due to size limitations.

---

## 🧠 Model Architecture

* Deep Learning-based model (CNN / Video Classification)
* Frame extraction from videos
* Feature learning using convolutional layers
* Classification into violence / non-violence
